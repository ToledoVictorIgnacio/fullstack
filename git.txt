

1. git init (inicia un repositorio local copiando todo lo que tengo en el directorio donde lo inicio)

2. U (sin seguimiento, no esta guardado en el repositorio local)

3. git add (agrega todos los datos al repositorio local)

    -git add index.html (agregar el archivo al repositorio local, agregando el nombre del archivo)

    -git add .(agrega todo el directorio donde estamos trabajando)

4. A (archivo agregado al repositorio local)

5. M (el archivo fue modificado, pero no guardado)

6. git commit (embalar el paquete de archivos para enviarlo)
    
    -m (comentar o describir la tarea realizada)

7. Si tocamos algo, volvemos a iniciar, menos el punto numero 1. 

8. git status (consulta el estado de git)

9. git no guarda directorios vacios por default

10. git remote -v (me muestra donde estan los codigos guardados en remoto)

11. git log (me muestra los commits hechos y quien los hizo)

12. creamos un repo en github

13. agregamos la url del repo remoto de git local - git remote add origin

14. enviamos el repo - git push origin master
